LƯU Ý: 
1/ Kiểm tra application.properties đã đúng với tên đăng nhập và mật khẩu vào sql chưa.

3/ Cho dù đặt tên database theo giống em hay theo ý mọi người thì đảm bảo hãy vào sql và tạo mới database đó. Rồi thay đổi tên kết nối nếu đặt tên theo sở thích
